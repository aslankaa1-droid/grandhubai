🚀 Как запустить локальный сервер

Для полной работы GrandHubAi (включая установку как приложение, оффлайн-режим, фоновые уведомления) нужно запустить локальный HTTP-сервер. Это занимает 10 секунд.

Способ 1 — Двойной клик (Windows)
  1. В корне папки проекта найдите файл start.bat
  2. Двойной клик по нему
  3. Откроется чёрное окно консоли + автоматически браузер на http://localhost:8080
Способ 2 — Двойной клик (macOS / Linux)
  1. В терминале: cd /путь/к/GrandHubAi
  2. chmod +x start.sh (один раз)
  3. ./start.sh
Способ 3 — Через Python вручную
cd "C:\Users\ais001\Documents\История_работы\GrandHubAi\GrandHubAi\src"
python -m http.server 8080

Затем откройте http://localhost:8080

Если Python не установлен

Скачайте с python.org/downloads → при установке отметьте «Add Python to PATH». После установки запустите start.bat снова.
