## 2. Реферат программы

**Наименование:** GrandHubAi — Multi-Agent AI Platform

**Год создания:** 2026

**Аннотация (200-1000 знаков):**

GrandHubAi — мульти-агентная платформа искусственного интеллекта, реализующая систему автоматического распределения пользовательских задач между специализированными промптами-агентами через главного координатора (Oracle). Программа представляет собой одностраничное web-приложение (PWA) на чистом JavaScript без внешних зависимостей. Поддерживает интеграцию с четырьмя провайдерами больших языковых моделей (Groq, OpenRouter, Anthropic, Ollama) через единый абстрактный интерфейс. Содержит библиотеку из 201 профессионального системного промпта в 27 предметных областях (машиностроение, медицина, право, литература, музыка, финансы и т.д.). Реализует декомпозицию пользовательских запросов, параллельное выполнение специалистами и синтез финального ответа. Обеспечивает локальное хранение данных в IndexedDB, работу в режиме PWA, поддержку голосового ввода и экспорта в JSON/Markdown/PDF.

**Объём программы:**
- Исходный код: ~250 КБ (приблизительно 8 000 строк JavaScript, HTML, CSS)
- Промпт-библиотека: 201 системный промпт (приблизительно 70 000 знаков русского текста)
- Документация: 8 файлов (приблизительно 100 000 знаков)

**Язык реализации:** JavaScript (ECMAScript 2020+), HTML5, CSS3

**Целевая платформа:** Современные web-браузеры (Chrome, Edge, Firefox, Safari, Yandex Browser) на любых операционных системах (Windows, macOS, Linux, Android, iOS)

**Назначение:** Профессиональный помощник для специалистов в инженерных, научных, медицинских, юридических, творческих и иных областях. Образовательный инструмент. Платформа для разработки специализированных AI-приложений.

## 4. Реферат для публикации в реестре

**(до 700 знаков, для опубликования)**

GrandHubAi — мульти-агентная платформа искусственного интеллекта на 201 профессиональном промпте в 27 предметных областях. Реализована как одностраничное web-приложение (PWA) на чистом JavaScript. Поддерживает четырёх провайдеров LLM (Groq, OpenRouter, Anthropic, Ollama) через единый интерфейс. Главный координатор Oracle декомпозирует пользовательские задачи и распределяет их между специалистами. Программа работает на любых устройствах (смартфонах, ПК), хранит данные локально в IndexedDB, поддерживает оффлайн-режим через Service Worker. Назначение: профессиональный помощник для инженеров, медиков, юристов, исследователей, творческих специалистов.

## 5. Указания на использование сторонних компонентов

В программе НЕ используются:
- Сторонние JavaScript-библиотеки (jQuery, React, Vue и т.д.)
- Внешние шрифты с CDN (Google Fonts и т.д.)
- Сторонние CSS-фреймворки (Bootstrap, Tailwind и т.д.)

Программа делает HTTP-запросы к внешним API-провайдерам (Groq, OpenRouter, Anthropic, Ollama) через стандартный fetch API браузера. Эти провайдеры не являются компонентами программы — программа лишь отправляет запросы по их публичным API.
