# SINTEM artefacts inside GrandHubAi repo

This directory mirrors selected artefacts from the SINTEM sprint kit
(`E:\Проекты Аслана\SINTEM\`) that are version-relevant alongside the
GrandHubAi codebase. The full sprint kit lives outside the repo.

Mirror policy: copy only artefacts that need git history (e.g. the
weekly Send-Kits a maintainer hands to a stakeholder). Working notes,
analysis, and large binaries stay outside the repo to keep clones small.

## Contents

- `sprint/week2_customer_dev/Send_Kit_Week2.html` — interactive
  recruit-and-track page for the 12 customer-development interviews
  with WB/Ozon sellers (week 2 of the SINTEM 30-day plan).
  Self-contained single-file HTML, autosaves to localStorage.
  Companion to `sprint/week2_customer_dev/{interview_script,
  target_channels, outreach_templates, interview_tracker}.md`
  in the source sprint kit.
