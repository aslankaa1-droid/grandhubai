# SINTEM Bot. Реплики

Все тексты, которые бот отправляет пользователю. Тон: деловой, на «вы», краткий, без эмодзи, без англицизмов, без декоративных тире. Длина оптимизирована под Telegram — большинство в 1–4 строки.

Каждый блок — готовая строка для копипаста в код. Ключи совпадают с теми, что в `bot_spec.md`.

Подстановки: `{{tg_first_name}}`, `{{plan}}`, `{{expires_at}}`, `{{counter_card_doctor}}`, и т. д. Шаблонизатор подставляет значения на стороне бекенда.

## 1. Onboarding и приветствия

### `welcome_short` (вариант A, формальный)

```
Здравствуйте. Это SINTEM — рабочий бот для селлеров Wildberries и Ozon.

Внутри пять помощников: разбор карточки, ответ на отзыв, скаут конкурентов, ABC/XYZ, юнит-экономика. Координирует Aida.

Первые пять запросов к каждому помощнику бесплатны. Регистрация не нужна.

Выберите помощника, чтобы начать.
```

### `welcome_short` (вариант B, теплее)

```
Здравствуйте, {{tg_first_name}}. Это SINTEM.

Помогаем продавцам WB и Ozon: разбираем карточки, отвечаем на отзывы, считаем юнит-экономику и сравниваем с конкурентами. Все ответы — от профильных AI-помощников.

Первые пять запросов к каждому помощнику бесплатны. Выбирайте.
```

### `start_returning`

```
С возвращением. Вот рабочее меню. Выберите помощника или откройте /profile, чтобы посмотреть остаток запросов.
```

### `agent_intro_card_doctor`

```
Карточка-доктор готов к работе.

Пришлите ссылку на карточку WB или Ozon, либо фото главной обложки и заголовок текстом. Я разберу слабые места по 12 пунктам и дам, что менять в первую очередь.
```

### `agent_intro_review_reply`

```
Помощник по ответам на отзывы готов.

Пришлите текст отзыва. Если знаете рейтинг — поставьте цифру в начале (например, «2 звезды:»). Можно прислать скриншот отзыва, я разберу.

Дам три варианта ответа: формальный, тёплый и краткий.
```

### `agent_intro_competitor_scout`

```
Скаут конкурентов готов.

Пришлите ссылку на свою карточку или артикул. Я найду 5–7 ближайших конкурентов и сравню по цене, рейтингу, отзывам и позиционированию.
```

### `agent_intro_abc_xyz`

```
Помощник по ABC/XYZ готов.

Пришлите выгрузку продаж за 3–6 месяцев в формате CSV или XLSX. Минимум: артикул, выручка, количество продаж по неделям. Я разнесу ассортимент по группам и подскажу, что чистить.
```

### `agent_intro_unit_econ`

```
Помощник по юнит-экономике готов.

Пришлите данные одной позиции: цена продажи, себестоимость, комиссия маркетплейса, логистика, упаковка, реклама. Можно текстом, можно файлом. Я посчитаю маржу, точку безубытка и пороги для скидок.
```

### `agent_intro_aida`

```
Aida на связи. Я координатор: подскажу, какой помощник подходит под вашу задачу, и переключу на него. Опишите задачу своими словами.
```

### `after_first_reply` (вариант A, нейтральный)

```
Ответ готов. Если нужно копнуть глубже — пришлите ещё данные тому же помощнику. Чтобы переключиться, откройте главное меню.
```

### `after_first_reply` (вариант B, мягкий апсейл)

```
Ответ готов. Бесплатно — пять таких разборов в день на каждого помощника. Если работаете много — откройте /profile и посмотрите тарифы, там кратно больше.
```

```

— Карточка-доктор: разбор карточки WB/Ozon
```

## 5. Edge cases

### `no_active_agent`

```
Сейчас не выбран ни один помощник. Откройте меню и выберите, с кем работать.
```

### `voice_not_supported`

```
Голосовые сообщения пока не обрабатываются. Пришлите запрос текстом — разберу.
```

### `photo_no_context`

```
Получил фото, но не понимаю, какому помощнику его отдать. Сначала выберите помощника в меню, потом пришлите фото.
```

### `doc_not_supported`

```
Этот помощник пока не работает с файлами. Пришлите данные текстом или сообщением со ссылкой.
```

### `file_too_large`

```
Файл больше 20 МБ. Telegram столько за раз не отдаёт. Пришлите выгрузку поменьше или поделите на части.
```

### `group_not_supported`

```
Бот работает только в личных чатах. Откройте диалог с ботом напрямую.
```

### `unsupported_input`

```
Не понял запрос. Пришлите текст, ссылку или фото с подписью.
```

### `rate_limited`

```
Слишком быстро. Подождите секунду и повторите.
```

## 6. Ошибки

### `err_timeout`

```
Ответ занимает дольше обычного. Сервис под нагрузкой. Попробуйте повторить через минуту, запрос засчитан не будет.
```

### `err_quota`

```
Сервис временно перегружен. Попробуйте через 10–15 минут. Если запрос срочный — напишите в поддержку, посмотрим, что можно сделать.
```

### `err_bad_output`

```
Не получилось собрать корректный ответ. Попробуйте переформулировать запрос или пришлите больше данных. Этот запрос с лимита не списан.
```

### `err_generic`

```
Что-то пошло не так на нашей стороне. Запрос с лимита не списан. Попробуйте ещё раз или напишите в поддержку.
```
