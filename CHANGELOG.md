## [1.2.0] — 2026-04-30 (вечер)

### 🎉 UX, мобильная адаптация, APK, автотесты

#### Добавлено
- **Enter — отправка сообщения**, Shift+Enter — новая строка, Ctrl+Enter — для совместимости
- **📎 Прикрепление файлов** — TXT, MD, JSON, CSV, log, XML, HTML, JS, Python, YAML, INI и изображения
  - До 5 МБ на файл, 20 МБ суммарно
  - Drag & drop в окно чата
  - Paste из буфера обмена (Ctrl+V)
  - Чипы прикреплённых файлов с возможностью удаления
  - Текстовые файлы встраиваются в запрос (с автоусечением до 30к символов)
- **📃 Экспорт в TXT** (plain text без markdown)
- **Mobile/Tablet UX:**
  - Touch-targets ≥ 44×44px (Apple HIG)
  - Safe-area-inset для iPhone notch / home indicator
  - viewport-fit=cover, apple-mobile-web-app-* мета-теги
  - iOS prevent zoom на input focus (font-size 16px+)
  - Modals снизу на мобильных (bottom sheet pattern)
  - Drawer-навигация для агентов на узких экранах
  - Альбомная ориентация компактнее
  - Hover-эффекты отключаются на touch-устройствах
  - 3 breakpoint уровня: 1024px / 900px / 480px / 360px
- **APK сборка:** новая папка `GrandHubAi-apk/`
  - Capacitor проект (package.json + capacitor.config.json) готов
  - 23 иконки приложения в нужных размерах (mipmap-*, adaptive, maskable, splash)
  - Web-исходники скопированы в `capacitor/www/`
  - BUILD_APK.md — 3 пути сборки (PWABuilder online, Capacitor local, Bubblewrap TWA)
  - PWABuilder инструкция за 10 минут без установки SDK
- **Автотесты Playwright:** новая папка `tests/`
  - 22 теста в headless Chromium
  - Покрытие: загрузка, промпты, темы, settings, чат, экспорт, безопасность, адаптивность
  - `npm test` запускает сервер + браузер + тесты автоматически

#### Исправлено
- 🐛 `setupVoice` мог упасть с null-reference если DOM не готов
- 🐛 `start.py` падал на Windows из-за cp1251 кодировки → utf-8 reconfigure
- 🐛 Неиспользуемая переменная `sentImages` в handleSend
- 🐛 Подсказка к прикреплениям: image-vision не реализован, теперь предупреждение для пользователя

#### Изменено
- README, CHANGELOG обновлены
- Поддержка `body.reduced-motion` для всех элементов через CSS

## [1.1.0] — 2026-04-30

### 🎉 Безопасность, темы, удобство запуска

#### Добавлено
- **CSP (Content Security Policy)** — строгая политика, блокирует внешние скрипты и eval
- **Frame-busting** — защита от clickjacking
- **AES-256-GCM шифрование** API-ключа master-паролем (Web Crypto API, PBKDF2 100k итераций)
- **6 новых тем:** Sepia, High Contrast (AAA по WCAG), Solarized Dark, Nord, Dracula, Auto (по системе)
- **4 размера шрифта** — S (14px) / M (16px) / L (18px) / XL (20px)
- **Reduced motion** — отключение анимаций для эпилепсии и слабых ПК
- **Циклическое переключение тем** через кнопку 🌓
- **Memory** — оркестратор передаёт последние 6 сообщений в LLM (бот помнит контекст)
- **file:// детект-баннер** с инструкцией запуска сервера
- **Стартеры:** `start.bat`, `start.sh`, `start.py` — однокнопочный запуск
- **Модал «Как запустить сервер»** с пошаговыми инструкциями
- **Endpoint-валидация** — проверка hostname matching при смене провайдера
- **Сохранение domainFilter** между сессиями (раньше сбрасывалось)
- **Updated current agent badge** — показывает режим работы (Oracle / команда / прямой вызов)

#### Исправлено
- 🐛 Жадный regex `\{[\s\S]*\}` мог поломать декомпозицию → заменён на balanced-braces extractor
- 🐛 Endpoint от старого провайдера мог использоваться после переключения
- 🐛 `applyProviderHints` экранировал HTML и сжирал ссылки на регистрацию
- 🐛 `handleStop` без inflight всё равно писал в лог
- 🐛 `currentAgentBadge` никогда не обновлялся
- 🐛 `domainFilter` после reload не восстанавливался
- 🐛 Опечатка «Раptor» (русская «а») в `aerospace.js`
- 🐛 Опечатка «AСCC» (русская «С») в `energy.js`
- 🐛 Меta description «175+» (вместо реальных 200+)
- 🐛 Документация ссылалась на «191 агента» (реально 201)

#### Изменено
- README, USER_GUIDE, INSTALL.md обновлены с инструкциями по стартерам
- SECURITY.md — полноценная политика безопасности с моделью угроз

## [1.0.0] — 2026-04-29

### 🎉 Первый релиз

#### Добавлено

**Ядро приложения**
- Single-file PWA на Vanilla JS (без зависимостей)
- 4 LLM-провайдера: Groq, OpenRouter, Anthropic Claude, Ollama
- Universal API через `callLLM()` с поддержкой OpenAI-compat и Anthropic стилей
- Streaming-ответы через Server-Sent Events
- Multi-agent оркестратор (Oracle decompose → parallel specialists → aggregate)
- Прямой вызов агента (минуя оркестратора)
- IndexedDB для хранения настроек, диалога, логов (с лимитом 200 / 100)
- AbortController для прерывания запросов
- Service Worker для оффлайн UI

**UI**
- Адаптивный дизайн (мобильные / планшеты / ПК)
- Темы dark/light с плавным переходом
- Боковая панель с иерархией агентов (по доменам)
- Поиск агентов
- Фильтр по доменам
- Модальное окно настроек
- Модальное окно с информацией об агенте (включая полный системный промпт)
- Activity Log с цветовой кодировкой (system / success / error)

**Безопасность**
- Защита от XSS через `escapeHtml()`
- Безопасный markdown rendering (без HTML-инъекций)
- Локальное хранение API-ключа в IndexedDB
- При экспорте JSON ключ заменяется на `[REDACTED]`
- Service Worker не кэширует API-запросы

**Голос**
- Web Speech API (распознавание русской речи)
- Fallback при отсутствии поддержки
- Error handling

**Экспорт**
- JSON (структурированный, с метаданными)
- Markdown (для GitHub/Notion/Obsidian)
- PDF (через `window.print()` + `@media print` стили)
